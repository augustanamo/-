CSS三栏布局左右固定，中间自适应

上次也考到了，在这里简单说一下可以实现的方法：
结构顺序：左 中 右
1.弹性布局：display: flex，两边固定宽度，中间flex：auto
2.定位：整体相对定位，三栏绝对定位，两边固定宽度，中间左右空出相应距离
结构顺序：中 左 右
3.圣杯布局：为了让中间div内容不被遮挡，将中间div设置了左右padding-left和padding-right后，将左右两个div用相对布局position: relative并分别配合right和left属性，以便左右两栏div移动后不遮挡中间div
4.双飞翼布局：为了让中间div内容不被遮挡，直接在中间div内部创建子div用于放置内容，在该div里用margin-left和margin-right为左右两栏div留出位置

Flex的属性

一、flex-direction: （元素排列方向）

二、flex-wrap: （换行内容一行容不下的时候才有效）

三、justify-content: （水平对齐方式）

四、align-items: （垂直对齐方式）

JS里有哪些数据类型？如何判断数据类型？

js中有六种数据类型，包括五种基本数据类型 （Number,String,Boolean,Undefined,Null）,和一种复杂数据类型（Object）。
typeof 操作符
由于js中的变量是松散类型的，所以它提供了一种检测当前变量的数据类型的方法，也就是typeof关键字.

typeof 123　　 //Number

typeof ‘abc’　　//String

typeof true //Boolean

typeof undefined //Undefined

typeof null //Object

typeof { } //Object

typeof [ ] //Object

typeof console.log() //Function

null类型进行typeof操作符后，结果是object，原因在于，null类型被当做一个空对象引用。

我们先来简单了解一下setTimeout延时器的运行机制。setTimeout会先将回调函数放到等待队列中，等待区域内其他主程序执行完毕后，按时间顺序先进先出执行回调函数。本质上是作用域的问题。因此若是这样将不会得到想要的结果输出1.2.3.4.5，而会连续输出5个6。

假如把var换成let，那么输出结果为1,2,3,4,5；
因为let i 的是区块变量，每个i只能存活到大括号结束，并不会把后面的for循环的 i 值赋给前面的setTimeout中的i；而var i 则是局部变量，这个 i 的生命周期不受for循环的大括号限制；

HTTP 301和401状态码

301：请求的网页已永久移动到新位置。服务器返回此响应（对 GET 或 HEAD 请求的响应）时，会自动将请求者转到新位置。
401：请求要求身份验证。对于登录后请求的网页，服务器可能返回此响应。
